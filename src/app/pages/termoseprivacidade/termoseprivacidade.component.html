Termos de Serviço e Política de Privacidade

LUMINA Tech

1. Sobre o Projeto

A LUMINA Tech é uma plataforma de e-commerce desenvolvida exclusivamente para fins educacionais e acadêmicos, como parte do Desafio Final da Trilha de Programação Front-End. O objetivo deste projeto é demonstrar competências técnicas em Angular, TypeScript e gestão de dados, sem fins comerciais reais.

2. Coleta e Armazenamento de Dados (LGPD)

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), informamos como seus dados são tratados nesta aplicação de demonstração:
Dados de Cadastro e Login: Ao criar uma conta ou fazer login, seu nome e e-mail são solicitados.
Como são armazenados: Esta aplicação utiliza a tecnologia LocalStorage do seu navegador. Isso significa que seus dados de cadastro não são enviados para um servidor externo ou banco de dados em nuvem sob controle da LUMINA Tech; eles ficam gravados apenas na memória do seu próprio dispositivo.
Formulário de Contato: Os dados inseridos na página de "Contato" (Nome, E-mail e Mensagem) são utilizados apenas para simular o envio de uma mensagem e demonstrar a interatividade da interface. Nenhuma mensagem é efetivamente enviada ou armazenada permanentemente.
Histórico de Pedidos: As informações visíveis em "Meus Pedidos" são geradas localmente para simular a experiência de compra.

3. Uso de Serviços de Terceiros

Para a exibição do catálogo de produtos, este site conecta-se à API pública DummyJSON. Esta conexão serve apenas para leitura de dados de produtos fictícios. Não compartilhamos seus dados pessoais de cadastro com este serviço.

4. Seus Direitos

Como titular dos dados, e considerando a arquitetura client-side (lado do cliente) deste projeto, você tem total controle sobre suas informações: Exclusão de Dados: Você pode remover todos os seus dados cadastrados neste site a qualquer momento limpando o "Cache" e o "Armazenamento Local" (LocalStorage) do seu navegador. Acesso: Você pode visualizar os dados armazenados acessando a área de perfil ou inspecionando o armazenamento do navegador.

5. Isenção de Responsabilidade (Termo de Uso)

Não Comercial: Este site não realiza vendas reais. Nenhum pagamento é processado e nenhum produto físico será entregue. Simulação: Preços, estoques e descrições de produtos são meramente ilustrativos.
